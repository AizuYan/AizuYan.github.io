selenium介绍

通过selenium，我们可以操作浏览器的行为，比如打开某个网页，在输入框中输入指定内容，点击某个按钮等，因此，我们可以通过它来实现部分web自动化测试，根据操纵者的操作，检测对应的反应是否满足我们的条件。

selenium安装

pip3 install -U selenium，通过这个命令安装selenium类库，-U表示如果已经安装则更新。等待安装成功之后，我们来测试下：

可以看到，Chrome浏览器是可以正常运行的，Firefox不行，这个提示我们要手动安装驱动，去https://doc.react-china.org/docs/components-and-props.html这个地方下载对应系统的驱动，然后放到$PATH中，确保系统能够找到他，再次运行，发现ok了，至此，我们的环境搭建成功了。
